ФАЙЛЫ ВИЗУАЛЬНОГО РЕДАКТОРА ПРОФИЛЕЙ НАГРУЗКИ LOCUST
======================================================

Основные файлы:
--------------
1. loadshape_editor.py
   - Главный файл визуального редактора
   - Графический интерфейс на tkinter
   - Визуализация графика на matplotlib
   - Парсинг и сохранение loadshape_stability.py

2. run_editor.sh
   - Скрипт для быстрого запуска редактора
   - Автоматическая проверка зависимостей
   - Установка matplotlib при необходимости

3. requirements-editor.txt
   - Список зависимостей для редактора
   - matplotlib>=3.5.0

Документация:
------------
4. EDITOR_GUIDE.md
   - Полное руководство по использованию редактора
   - Описание всех функций
   - Примеры профилей нагрузки
   - Советы и best practices

5. INSTALL.md
   - Инструкции по установке
   - Устранение типичных проблем
   - Системные требования
   - Альтернативные методы установки

6. EDITOR_README.md (в корне проекта)
   - Краткая инструкция по запуску
   - Скриншот интерфейса
   - Список возможностей

Дополнительные файлы:
-------------------
7. loadshape_templates.py
   - 10 готовых шаблонов профилей нагрузки
   - Описания и use cases для каждого шаблона
   - Функция вывода информации о шаблонах

8. README.md (обновлён)
   - Добавлен раздел о визуальном редакторе
   - Инструкции по установке и запуску
   - ASCII-скриншот интерфейса

Целевые файлы (редактируются):
-----------------------------
9. loadshape_stability.py
   - Профиль нагрузки для Locust
   - Редактируется визуальным редактором
   - Сохраняет структуру и комментарии

Структура проекта:
-----------------
fastapi_portfolio/
├── loadshape_editor.py          # Главный редактор
├── run_editor.sh                # Скрипт запуска
├── requirements-editor.txt      # Зависимости
├── EDITOR_GUIDE.md              # Полное руководство
├── INSTALL.md                   # Инструкции по установке
├── loadshape_templates.py       # Шаблоны профилей
├── loadshape_stability.py       # Целевой файл (редактируется)

Корень проекта:
├── EDITOR_README.md             # Краткая инструкция
└── fastapi_portfolio/           # Директория с редактором

Использование:
-------------
1. Установка:
   cd fastapi_portfolio
   pip install -r requirements-editor.txt

2. Запуск:
   ./run_editor.sh
   # или
   python3 loadshape_editor.py

3. Просмотр шаблонов:
   python3 loadshape_templates.py

4. Документация:
   cat EDITOR_GUIDE.md
   cat INSTALL.md
