## 🎓 Примеры использования

### Создание простого профиля

1. Запустите редактор: `./run_editor.sh`
2. Нажмите "➕ Добавить"
3. Заполните параметры:
   - Длительность: 60 секунд
   - Пользователи: 10
   - Spawn rate: 1
   - User Classes: WebUser_UC01
4. Нажмите "✅ Сохранить"
5. Посмотрите график
6. Нажмите "💾 Сохранить в файл"

### Редактирование существующего этапа

1. Выберите этап в списке
2. Нажмите "✏️ Редактировать"
3. Измените параметры
4. Нажмите "✅ Сохранить"
5. График обновится автоматически

### Использование шаблона

1. Откройте `loadshape_templates.py`
2. Скопируйте нужный шаблон
3. Вставьте в `loadshape_stability.py`
4. Откройте редактор для визуализации
5. Настройте под свои нужды

## 🐛 Известные ограничения

1. Редактор работает только с форматом Locust LoadShape
2. Требуется установка matplotlib и tkinter
3. Изменяет только блок `stages` в файле
4. Не поддерживает комментарии внутри stages
